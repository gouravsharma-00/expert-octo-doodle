# 🚀 AI-Driven Bug Detection & Intelligent Code Debugger

An AI-powered web application that automatically detects bugs in code, explains errors in simple language, and suggests intelligent fixes. This tool acts as a smart debugging assistant for developers, helping them write better and error-free code.

---

## 📌 Features

* 🔍 **Automatic Bug Detection**
  Detects syntax, logical, and runtime errors in code.

* 💡 **Smart Fix Suggestions**
  Provides AI-generated solutions with explanations.

* 📘 **Beginner-Friendly Explanations**
  Explains errors in simple and easy language.

* 💻 **Multi-Language Support**
  Supports Java, Python, and JavaScript.

* ⚡ **Real-Time Code Analysis**
  Instantly analyzes code and highlights issues.

* 🧠 **AI-Powered Debugging**
  Uses advanced AI models (Gemini/OpenAI) for deep code understanding.

* 🎨 **Interactive Code Editor**
  Built with Monaco Editor (VS Code-like experience).

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/AI-Code-Debugger.git
cd AI-Code-Debugger
```

### 2️⃣ Install dependencies

#### Frontend

```
cd frontend
npm install
npm start
```

#### Backend

```
cd backend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the backend folder and add:

```
PORT=5000
GEMINI_API_KEY=your_api_key_here
```

---

## 🧠 How It Works

1. User writes or uploads code in the editor
2. Code is sent to the backend server
3. Backend sends code to AI API
4. AI analyzes and returns:

   * Errors
   * Fix suggestions
   * Explanations
5. Results are displayed in the UI
